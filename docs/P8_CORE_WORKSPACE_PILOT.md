# P8 Core Workspace Pilot

`@tifa-assistant/core` is the first workspace pilot package.

## What Changed

- Added `packages/core`.
- Package re-exports the `lib/tifa-core` boundary.
- Added package validation scripts.
- Package can be built and typechecked independently.

## What Did Not Change

- Source of truth remains `lib/tifa-core`.
- Next.js app remains local-first.
- Compatibility imports remain supported.
- SaaS mode is not enabled.
- Other packages are not published.

## Publish Target

`@tifa-assistant/core`
