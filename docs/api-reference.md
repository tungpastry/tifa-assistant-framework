# API Reference

The Next.js backend exposes several endpoints used by the frontend.

## `GET /api/today`
Returns the daily generated vibe and mood data.
*   **Response:** JSON object containing mood, quote, and paths to audio files.
*   **Source:** Reads from `runtime/latest.json`.

## `GET /api/playlist?mood={mood}`
Returns curated YouTube and Spotify links for the specified mood.
*   **Query Params:** `mood` (string, default: "focused").
*   **Response:** JSON object containing arrays of Spotify and YouTube track/playlist URLs.

## `POST /api/tifa`
Real-time chat endpoint interacting with the local Ollama model.
*   **Body:** `{ "messages": [...] }`
*   **Response:** Streaming text response from the LLM.

## `GET /api/voice?text={text}`
Real-time Text-to-Speech endpoint.
*   **Query Params:** `text` (string, the text to synthesize).
*   **Response:** Audio stream (WAV) generated by local Piper TTS.
