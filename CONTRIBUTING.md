# Contributing to TradeVibe

We love your input! We want to make contributing to this project as easy and transparent as possible.

## Development Setup
1. Fork the repo and clone it locally.
2. Run `npm ci` to install dependencies.
3. Copy `.env.example` to `.env` and configure your local AI tools.
4. Run `npm run dev` to start the development server.

## Pull Request Process
1. Create a new branch (`feature/your-feature` or `bugfix/issue-name`).
2. Make your changes.
3. Ensure the code passes linting: `npm run lint`.
4. Update the `CHANGELOG.md` with your changes.
5. Submit a Pull Request targeting the `main` branch.

## Commit Guidelines
We use conventional commits (e.g., `feat: add new player`, `fix: resolve audio bug`, `docs: update readme`).
